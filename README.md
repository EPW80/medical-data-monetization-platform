# Health Data Monetization Backend

A blockchain-based backend service for securely monetizing health data using Ethereum smart contracts. This project enables users to register, purchase, and manage access to health data while maintaining privacy and security through blockchain technology.

## Features

- Decentralized data storage and management
- Access control using Ethereum smart contracts
- Cryptocurrency payment system
- RESTful API endpoints for data submission
- Support for multiple Ethereum networks (local development and Sepolia testnet)
